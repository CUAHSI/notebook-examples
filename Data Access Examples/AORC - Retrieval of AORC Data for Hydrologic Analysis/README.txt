Jupyter Notebooks for the Retrieval of AORC Data for Hydrologic Analysis

This resource contains the following Jupyter Notebooks and  python files with functions called by the notebooks.
AORC_Point_Data_Retrieval.ipynb
AORC_Zone_Data_Retrieval.ipynb
This resource is designed to be opened from a JupyterHub platform connected to CUAHSI HydroShare and has been tested using CUAHSI JupyterHub, CyberGIS-Jupyter for Water and CIROH JupyterHub.

At this resource in HydroShare use Open With on any of these application servers to launch into the respective JupyterHub environment.  Then open the Jupyter Notebook for the format of the data you wish to retrieve and follow the instructions it contains.
